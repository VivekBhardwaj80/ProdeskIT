# Prodesk IT Landing Page

## Project Overview

This project is a responsive landing page developed as part of the Prodesk IT Sprint 01 assignment. The goal of the project is to demonstrate strong frontend fundamentals using HTML, CSS, and JavaScript without relying on external UI frameworks.

The landing page includes a responsive navigation bar, hero section, about section, services section, contact section, and footer. It is designed to provide a clean user experience across desktop and mobile devices.

---

## Features

* Responsive Navigation Bar
* Sticky Navbar
* Hero Section with Call-to-Action Buttons
* About Section
* Services Section using CSS Grid
* Contact Section
* Footer with Social Media Links
* Dark/Light Mode Toggle
* Hover Animations and Card Effects
* Mobile-Friendly Layout
* Smooth Scrolling Navigation

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Flexbox
* CSS Grid
* Media Queries
